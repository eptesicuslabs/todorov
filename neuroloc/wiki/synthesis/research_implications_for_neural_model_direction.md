# research implications for neural model direction

status: current (as of 2026-04-23).

## headline

this research pass does not reactivate experiments. it does sharpen the backlog. the strongest center of gravity is now:

- multi-timescale state
- indexing plus replay
- routing and gating as first-class control primitives
- branch or compartment computation as a serious future direction
- latent state-action models as the strongest external ai import

## ranked implications

### actionable now

- keep phase 1 framed as state plus action rather than next-token prediction
- keep replay, indexing, routing, and bounded memory explicit in backlog theory
- treat latent state learning and latent planning as the primary external-ai comparison frame

### simulation next

- strengthen replay-benefit and offline-reuse probes
- add routing-sensitive probes where control and access matter
- keep latent-rollout and hard-case extra-compute probes central

### worth preserving but not acting on yet

- glia-inspired slow control planes
- compartment or branch-local compute as a model design direction
- self-modeling as attention and agency monitoring

### interesting but too weak

- direct trainable-delay imports from myelination
- active inference as the whole engineering stack
- any claim that one external architecture family already solves the project

## effect on canonical project state

the curriculum remains the only active workstream. the architecture backlog becomes clearer, not more urgent. no result here authorizes paid compute. what changes is the theory base the backlog will restart from later.
