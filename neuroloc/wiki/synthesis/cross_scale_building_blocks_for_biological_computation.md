# cross scale building blocks for biological computation

status: current (as of 2026-04-23).

## thesis

the most useful unifying picture from this research pass is not one named mechanism. it is a reusable primitive set:

- compete
- gate
- recur
- route
- index
- replay
- compress
- stabilize

biology appears to reuse these moves across synapses, cells, circuits, and whole cognitive systems.

## the primitive bundle

### competition

competition reduces overlap and interference. divisive normalization, sparse coding, and inhibitory selection are all versions of this move.

### gating

gating decides which path is open now. disinhibition, thalamic relay control, and modulatory write-mode switches all fit here.

### recurrence

recurrence maintains or refines state after the first pass. this includes reverberation, pattern completion, and feedback correction.

### routing

routing determines which parts of the fixed graph communicate effectively now. synchrony, thalamic coordination, and gain control all point here.

### indexing

indexing stores a compact handle that can later drive reinstatement or completion.

### replay

replay is the offline or delayed reuse of prior activity to consolidate, compose, or prepare behavior.

### compression

compression is not only about energy. it is also about address formation and interference control.

### stabilization

stabilization prevents the rest of the system from saturating or collapsing. homeostasis is not optional polish; it is part of the compute stack.

## what this changes

this primitive set is a better design vocabulary for neuroloc than a long list of paper names. it should guide new simulations, new synthesis pages, and future architecture hypotheses.
