# results

this directory is a historical summary layer for neuroloc experiments.

it is not the canonical state surface and it is not a complete live index of
the current simulation battery. for current state, use:

- `neuroloc/wiki/PROJECT_PLAN.md`
- `neuroloc/wiki/tests/`
- `neuroloc/simulations/suite_registry.py`
- `pdf_curriculum/index/curriculum_status.md`

## current interpretation

the early entries here document the first wave of neuroloc pilot experiments:

- pattern completion baseline
- competitive-selection support recovery
- leak versus carry retention tradeoff
- activity-dependent forgetting pilot
- geometric-product control pilot
- capacity and compression pilots
- sparse-topology and forward-learning probes

those results remain useful as historical evidence, but they no longer define
the full project surface.

## active backlog restart point

when the architecture backlog resumes after the curriculum, it restarts from:

- the implemented `biology_phase1` symbolic battery
- the remaining latent-world deliberation probe
- the model-side neural-model evaluation surface

the decisive archived finding is that the last paid neural-machine branch
fit the byte distribution but still failed retrieval across six paid runs.
the canonical diagnosis is in
`neuroloc/wiki/synthesis/substrate_requires_architectural_change.md`.

## scope rule

add historical result summaries here only when they help humans browse the
archive. do not treat this directory as the canonical source of truth.
